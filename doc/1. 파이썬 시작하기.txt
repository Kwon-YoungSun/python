# day01
표현식	: 어떠한 값을 만들어 내는 간단한 코드
문장	: 표현식이 하나 이상 모인 것
프로그램	: 문장이 모인 것
키워드	: 특별한 의미가 부여된 단어. 예약어

참고 ] 파이썬은 대소문자를 구분하는 프로그래밍 언어이다
-----------------------------------------------------------------------------------------------------
1. 식별자(identifier) : 프로그래밍 언어에서 이름을 붙일 때 사용하는 단어
- 키워드를 사용하면 안된다.
- 특수 문자는 언더바( _ ) 만 허용
- 숫자로 시작하면 안된다.
- 공백을 포함할 수 없다.
- 알파벳을 사용하는 것이 관례이다.
- 의미 있는 단어를 사용하자

참고 ] 스네이크 케이스와 캐멀 케이스
	1) 스네이크 케이스
		단어 사이에 언더바( _ ) 를 붙여 식별자를 만든다.
			itemlist ==> item_list
		함수, 변수에 사용
	2) 캐멀 케이스
		itemlist ==> ItemList
		클래스에 사용

참고 ] 식별자 구분하기
- 파이썬에서 캐멀 케이스의 경우 '첫번째 글자를 소문자로 적는다'는 사용되지 않음
	PrintHello ==> 파이썬에서 사용
	printHello ==> 파이썬에서 사용하지 않음

- 첫번째 글자가 소문자이다. ==> 무조건 스네이크 케이스
- 첫번째 글자가 대문자이다. ==> 무조건 캐멀 케이스

	예) 1 + 1, 10 - 10
